Lista di comandi da eseguire per installare, configurare e usare sass sul proprio ambiente :

  1. Installare NodeJs, potete recarvi qui e scaricare l’ultima versione (LTS) consigliata
  2. – Npm: è il principale software utilizzato per maneggiare e installare moduli NodeJS
    – Node-sass: è una libreria che permette di compilare nativamente i file .scss in css ad una velocità incredibile
    Verificare di aver installato nodeJs entrando nella SHELL e digitando il comando :

    node -v,


    successivamente installare NODE-SASS :
  3. Per installare a livello globale Node-sass esegui il comando da SHELL :

    npm install node-sass -g


    Alla fine dell’installazione verificare sempre che sia installato tutto correttamente eseguendo da SHELL il comando :

    node-sass -v


  4. Per verificare il corretto funzionamento, creiamo un progetto di prova strutturato nel seguente modo (scrivete anche il codice che vedete nell’immagine successiva nei file opportuni) :

  5. A questo punto andremo ad includere nel nostro file index.html il contenuto generato e convertito in css dei file .sass e .scss, da SHELL eseguendo i comandi :
  6. script per convertire tutti i file dalla folder scss con estensioni .scss nella folder css :

    node-sass scss -o css

  7. script per convertire tutti i file dalla folder sass con estensioni .saas nella folder css :

    node-sass sass -o css

  8. oppure creare un file batch salvandolo con estensione .sh inserendo il seguente script :

    #!/bin/sh
    echo "Creazione script per convertire tutti i file dalla folder scss con estensioni .scss nella folder css"
    node-sass scss -o css
    echo "Creazione script per convertire tutti i file dalla folder sass con estensioni .saas nella folder css"
    node-sass sass -o css
